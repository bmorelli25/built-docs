List Config Store Option

List all existing Config Store Option.

Request

GET /api/v1/platform/configuration/store

Responses

200

(ConfigStoreOptionList) List of existing Config Store Option

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request example

curl -XGET https://{{hostname}}/api/v1/platform/configuration/store \
-H "Authorization: ApiKey $ECE_API_KEY"
