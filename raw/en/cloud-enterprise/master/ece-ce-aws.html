AWS

If you use Amazon Web Services (AWS) and Amazon Elastic Compute Cloud (EC2), you can install Elastic Cloud Enterprise by using the preconfigured Elastic AMIs that can be deployed from the AWS Management Console.

To learn more about AMIs, see Amazon Machine Images (AMI).
