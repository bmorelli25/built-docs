Considerations for basepathedit

In dev mode, Kibana by default runs behind a proxy which adds a random path component to its URL.

You can set this explicitly using server.basePath in Configuring Kibana.

Use the server.rewriteBasePath setting to tell Kibana if it should remove the basePath from requests it receives, and to prevent a deprecation warning at startup. This setting cannot end in a slash (/).

If you want to turn off the basepath when in development mode, start Kibana with the --no-basepath flag

yarn start --no-basepath
