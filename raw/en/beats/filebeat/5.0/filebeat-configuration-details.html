Configuration Options (Reference)edit

Before modifying configuration settings, make sure you’ve completed the configuration steps in the Getting Started.

The Filebeat configuration file, filebeat.yml, uses YAML for its syntax. The configuration options are described in the following sections. After changing configuration settings, you need to restart Filebeat to pick up the changes.

  • Console Output Configuration
  • SSL Configuration
  • Paths Configuration
  • Logging Configuration
