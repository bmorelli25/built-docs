MySQL moduleedit

This module collects and parses the slow logs and error logs created by MySQL.

Compatibilityedit

The MySQL module was tested with logs from versions 5.5 and 5.7.

On Windows, the module was tested with MySQL installed from the Chocolatey repository.

Dashboardedit

This module comes with a sample dashboard.

Error logs fileset settingsedit

var.pathsedit

An array of paths where to look for the log files. If left empty, Filebeat will choose the paths depending on your operating systems.
