Breaking changesedit

Affecting all Beats

  • Fix primitive types conversion if values are read from environment variables. 2698

Metricbeat

Packetbeat

  • Fix the bpf_filter setting. 2660

Filebeat

  • Fix input buffer on encoding problem. 2416
