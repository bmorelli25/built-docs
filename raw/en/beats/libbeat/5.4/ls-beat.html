Example: Building Lsbeat from Scratchedit

Lsbeat is similar to the ls command-line tool, but instead of printing the files and subdirectories to the screen, Lsbeat periodically ships them to Elasticsearch for storage.

To help you learn how to build a Beat, we’ve created this blog post that describes how to build Lsbeat from scratch. You can refer to the Lsbeat implementation as a working example.
