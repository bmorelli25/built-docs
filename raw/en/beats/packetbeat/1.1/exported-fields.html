Exported Fieldsedit

This document describes the fields that are exported by Packetbeat for each transaction. They are grouped in the following categories:

  • Event Fields
  • ICMP Fields
  • DNS Fields
  • Http Fields
  • Memcache Fields
  • Mysql Fields
  • PostgreSQL Fields
  • Thrift-RPC Fields
  • Redis Fields
  • MongoDb Fields
  • Measurements Fields
  • Environmental Fields
  • Raw Fields
