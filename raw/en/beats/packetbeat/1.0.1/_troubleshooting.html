Troubleshootingedit

If you suspect an issue with Packetbeat, please read the following tips and send us the logs and, if possible, a traffic trace.

Please contact us on the forums, and we’ll help you troubleshoot the problem.

If you’re sure you found a bug, you can open a ticket on GitHub. Note, however, that we close GitHub issues containing questions or requests for help if they don’t indicate the presence of a bug.
