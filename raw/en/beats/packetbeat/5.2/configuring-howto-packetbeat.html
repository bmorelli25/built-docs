Configuring Packetbeatedit

After following the configuration steps in the Getting Started, you might want to fine tune the behavior of Packetbeat. This section describes some common use cases for changing configuration options.

To configure Packetbeat, you edit the configuration file. For rpm and deb, you’ll find the default configuration file at /etc/packetbeat/packetbeat.yml. There’s also a full example configuration file at /etc/packetbeat/packetbeat.full.yml that shows all non-deprecated options. For mac and win, look in the archive that you extracted.
