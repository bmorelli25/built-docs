Common fieldsedit

Contains common fields available in all event types.

metricset.moduleedit

The name of the module that generated the event.

metricset.nameedit

The name of the metricset that generated the event.

metricset.hostedit

Hostname of the machine from which the metricset was collected. This field may not be present when the data was collected locally.

metricset.rttedit

type: long

required: True

Event round trip time in microseconds.

metricset.namespaceedit

type: keyword

Namespace of dynamic metricsets.

typeedit

example: metricsets

required: True

The document type. Always set to "metricsets".
