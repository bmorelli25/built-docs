agentedit

agent Metricset fetches information from a Consul Agent in Client mode. It fetches information about the health of the autopilot, runtime metrics and raft data.

Example configurationedit

The consul module supports the standard configuration options that are described in Specify which modules to run. Here is an example configuration:

metricbeat.modules:
- module: consul
  metricsets:
  - agent
  enabled: true
  period: 10s
  hosts: ["localhost:8500"]
