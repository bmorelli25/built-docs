System fsstat metricsetedit

The System fsstat metricset provides overall file system statistics.

This metricset is available on:

  • FreeBSD
  • Linux
  • macOS
  • OpenBSD
  • Windows

Configurationedit

filesystem.ignore_types - A list of filesystem types to ignore. Metrics will not be collected from filesystems matching these types. This setting also affects the filesystem metricset.

Fieldsedit

For a description of each field in the metricset, see the exported fields section.
