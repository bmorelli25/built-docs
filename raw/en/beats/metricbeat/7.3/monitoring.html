Monitoring Metricbeatedit

You can use the Elastic Stack monitoring features to gain insight into the health of Metricbeat agents running in your environment.

To monitor Metricbeat, make sure monitoring is enabled on your Elasticsearch cluster, then configure the method used to collect Metricbeat metrics. You can use one of following methods:

  • Internal collection
  • Metricbeat collection

To learn about monitoring in general, see Monitoring the Elastic Stack.
