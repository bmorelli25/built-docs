System CPU Metricsetedit

The System cpu metricset provides CPU statistics.

This metricset is available on:

  • Darwin
  • FreeBSD
  • Linux
  • OpenBSD
  • Windows

Fieldsedit

For a description of each field in the metricset, see the exported fields section.
