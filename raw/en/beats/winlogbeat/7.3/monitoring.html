Monitoring Winlogbeatedit

You can use the Elastic Stack monitoring features to gain insight into the health of Winlogbeat agents running in your environment.

To monitor Winlogbeat, make sure monitoring is enabled on your Elasticsearch cluster, then configure the method used to collect Winlogbeat metrics. You can use one of following methods:

  • Internal collection
  • Metricbeat collection

To learn about monitoring in general, see Monitoring the Elastic Stack.
