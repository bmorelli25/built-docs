Delete API keys

Delete or invalidate API keys.

Request

DELETE /api/v1/users/auth/keys

Request body

(DeleteApiKeysRequest) (required) The request to delete API keys

Responses

200
(EmptyResponse) The API keys are deleted.
449

(BasicFailedReply) Elevated permissions are required. (code: root.unauthorized.rbac.elevated_permissions_required)

Headers

x-cloud-error-codes (string; allowed values: [root.unauthorized.rbac.elevated_permissions_required])
The error codes associated with the response

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request example

curl -XDELETE https://api.elastic-cloud.com/api/v1/users/auth/keys \
-H "Authorization: ApiKey $CLOUD_API_KEY" \
-H 'Content-Type: application/json' \
-d '
{
   "keys" : [
      "string"
   ]
}
'

Get all API keys

Retrieves the metadata for all of the API keys that the user generated.

Request

GET /api/v1/users/auth/keys

Responses

200
(ApiKeysResponse) The metadata for the API keys is retrieved.

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request example

curl -XGET https://api.elastic-cloud.com/api/v1/users/auth/keys \
-H "Authorization: ApiKey $CLOUD_API_KEY"

Create API key

Creates a new API key.

Request

POST /api/v1/users/auth/keys

Request body

(CreateApiKeyRequest) (required) The request to create the API key

Responses

201
(ApiKeyResponse) The API key is created and returned in the body of the response.
400

(BasicFailedReply) The request is invalid. Specify a different request, then try again. (code: api_keys.invalid_input)

Headers

x-cloud-error-codes (string; allowed values: [api_keys.invalid_input])
The error codes associated with the response
449

(BasicFailedReply) Elevated permissions are required. (code: root.unauthorized.rbac.elevated_permissions_required)

Headers

x-cloud-error-codes (string; allowed values: [root.unauthorized.rbac.elevated_permissions_required])
The error codes associated with the response

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request example

curl -XPOST https://api.elastic-cloud.com/api/v1/users/auth/keys \
-H "Authorization: ApiKey $CLOUD_API_KEY" \
-H 'Content-Type: application/json' \
-d '
{
   "authentication_token" : "string",
   "description" : "string"
}
'
