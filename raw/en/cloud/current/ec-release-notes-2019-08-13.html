AWS region EU (London) available

Elasticsearch Service is now available in the AWS region EU (London), also known as eu-west-2. You can select the new region when you create new deployments.

Like other recently added regions, EU (London) supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.

Service release: August 13, 2019 (Milestone 26)
