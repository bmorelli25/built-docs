Azure region Japan East (Tokyo) available

Elasticsearch Service is now available in the Azure region Japan East (Tokyo). You can select the new region when you create new deployments.

Like other recently added regions, Tokyo support only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.

Service release: February 12, 2020
