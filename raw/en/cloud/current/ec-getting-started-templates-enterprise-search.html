Enterprise Search template

This template provides the capabilities of Elastic Enterprise Search, including:

  • App Search - deliver a pre-tuned search experience in your applications
  • Workplace Search - deliver a fast and relevant search experience across your organization

To enable Enterprise Search on Elasticsearch Service, see Enable Enterprise Search.

For full Enterprise Search documentation, see the Enterprise Search Reference.

Included in this template:

Amazon Web Services (AWS)
  • Elasticsearch data nodes: Start at 1 GB memory x 1 availability zone. Hosted on AWS m5 instances.
  • Enterprise Search nodes: Start at 4 GB memory x 2 availability zones. Hosted on AWS m5 instances.
  • Kibana: Starts at 1 GB memory x 1 availability zone. Hosted on AWS r4 instances.
Google Cloud Platform (GCP)
  • Elasticsearch data nodes: Start at 1 GB memory x 1 availability zone. Hosted on custom CPU-optimized GCP instances.
  • Enterprise Search nodes: Start at 4 GB memory x 2 availability zones. Hosted on custom CPU-optimized GCP instances.
  • Kibana: Starts at 1 GB memory x 1 availability zone. Hosted on custom memory-optimized GCP instances.
Microsoft Azure
  • Elasticsearch data nodes: Start at 1 GB memory x 1 availability zone. Hosted on Azure D64sv3 instances.
  • Enterprise Search nodes: Start at 4 GB memory x 2 availability zones. Hosted on Azure D64v3 instances.
  • Kibana: Starts at 1 GB memory x 1 availability zone. Hosted on Azure E32sv3 instances.
