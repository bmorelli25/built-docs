User Features Controls

  • Get features

Get features

Gets the restricted features current user has access.

Request

GET /api/v1/users/feature-controls

Responses

200
(UserFeatures) The request was successful and user features were returned.

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request example

curl -XGET https://api.elastic-cloud.com/api/v1/users/feature-controls \
-H "Authorization: ApiKey $CLOUD_API_KEY"
