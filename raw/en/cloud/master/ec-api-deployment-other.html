Other deployment operations

Restart an Elasticsearch cluster

Restart a cluster.

curl -XPOST \
-H "Authorization: ApiKey $CLOUD_API_KEY" \
"https://api.elastic-cloud.com/api/v1/regions/$REGION_ID/clusters/elasticsearch/$CLUSTER_ID/_restart"

Shut down an Elasticsearch Service cluster

Shut down an Elasticsearch Service cluster.

curl -XPOST \
-H "Authorization: ApiKey $CLOUD_API_KEY" \
"https://api.elastic-cloud.com/api/v1/regions/$REGION_ID/clusters/elasticsearch/$CLUSTER_ID/_shutdown"
