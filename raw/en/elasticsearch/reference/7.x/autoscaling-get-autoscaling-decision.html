Get autoscaling decision APIedit

Get autoscaling decision.

Requestedit

GET /_autoscaling/decision/

Prerequisitesedit

  • If the Elasticsearch security features are enabled, you must have manage_autoscaling cluster privileges. For more information, see Security privileges.

Descriptionedit

This API gets the current autoscaling decision based on the configured autoscaling policy. This API will return whether or not autoscaling is needed.

Examplesedit

This example retrieves the current autoscaling decision.

GET /_autoscaling/decision

The API returns the following result:

{
  decisions: []
}
