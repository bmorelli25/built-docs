Command line toolsedit

Elasticsearch provides the following tools for configuring security and performing other tasks from the command line:

  • elasticsearch-certgen
  • elasticsearch-certutil
  • elasticsearch-croneval
  • elasticsearch-node
  • elasticsearch-saml-metadata
  • elasticsearch-setup-passwords
  • elasticsearch-shard
  • elasticsearch-syskeygen
  • elasticsearch-users
