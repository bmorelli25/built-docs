Elasticsearch version 7.8.0edit

Coming in 7.8.0.

Breaking changesedit

Search
  • Scripts used in value_count will now receive a number if they are counting a numeric field and a GeoPoint if they are counting a geo_point fields. They used to always receive the String representation of those values. #54854
Features/Indices APIs
  • The privilege named indices:admin/create will no longer allow the auto creation of indices. Use create_index instead. #55858
