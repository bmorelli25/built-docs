Elastic Stack 7.2.0 released

Elasticsearch Add-On for Heroku now supports the latest version of the Elastic Stack, which includes Elasticsearch and Kibana 7.2.0.

For more information about the features in Elasticsearch and Kibana 7.2.0, see:

  • Elasticsearch 7.2.0 Release Notes
  • Kibana 7.2.0 Release Notes

Service release: June 25, 2019
