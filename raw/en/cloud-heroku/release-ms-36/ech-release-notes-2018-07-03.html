Elasticsearch Service branding update

We updated the Elastic Cloud brand to encompass all cloud-based offerings from Elastic. If it’s a software-as-a-service (SaaS) offering from us, it now falls under the Elastic Cloud umbrella.

Here’s the difference:

Old
Elastic Cloud = Hosted Elasticsearch + Kibana
New
Elastic Cloud = Elasticsearch Service + Elastic App Search Service + Elastic Site Search Service

The documentation you are currently reading has been updated to refer to the Elasticsearch Service in line with this branding change. To learn more about the different products that fall under the Elastic Cloud umbrella, see Elastic Cloud: Hosted Elasticsearch, Hosted Search.

Service update: July 3, 2018
