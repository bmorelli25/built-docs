Elastic Stack 7.3.1 released

Elasticsearch Add-On for Heroku now supports the latest version of the Elastic Stack, which includes Elasticsearch and Kibana 7.3.1.

For more information about the features in Elasticsearch and Kibana 7.3.1, see:

  • Elasticsearch 7.3.1 Release Notes
  • Kibana 7.3.1 Release Notes
