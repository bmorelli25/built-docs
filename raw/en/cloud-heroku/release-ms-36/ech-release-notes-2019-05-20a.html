Elastic Stack 6.8.0 released

Elasticsearch Add-On for Heroku now supports the latest version of the Elastic Stack, which includes Elasticsearch and Kibana 6.8.0.

For more information about the features in Elasticsearch and Kibana 6.8.0, see:

  • Elasticsearch 6.8.0 Release Notes
  • Kibana 6.8.0 Release Notes

Service release: May 20, 2019
