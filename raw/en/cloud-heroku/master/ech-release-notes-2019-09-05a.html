Elasticsearch and Kibana 6.8.3 available on Elasticsearch Add-On for Heroku

Elasticsearch Add-On for Heroku now supports the 6.8.3 of the Elastic Stack, which includes Elasticsearch and Kibana. 6.8.3 is a bug fix release with notable improvements for snapshot and restore on Azure.

For more information about the features in Elasticsearch and Kibana 6.8.3, see:

  • Elasticsearch 6.8.3 Release Notes
  • Kibana 6.8.3 Release Notes

Service release: September 5, 2019
