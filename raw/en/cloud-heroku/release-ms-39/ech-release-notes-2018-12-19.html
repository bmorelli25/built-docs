Elastic Stack 6.5.4 released

Elasticsearch Add-On for Heroku now supports the latest version of the Elastic Stack, which includes Elasticsearch and Kibana 6.5.4.

For more information about the features in Elasticsearch and Kibana 6.5.4, see:

  • Elasticsearch 6.5.4 Release Notes
  • Kibana 6.5.4 Release Notes

Service release: December 19, 2018
