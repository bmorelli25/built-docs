Minor Cloud Console updates released

In preparation for offering more flexible deployment configurations for different use cases, we released some user interface changes that you might notice. These changes include references to tags such as highio.classic or lowio.classic. You do not need to take any further action as a result of these UI changes and no additional functionality is available.

These tags refer to the virtualized hardware resources or instance configurations that your Elasticsearch nodes and Kibana instances run on. We use these labels to make sure that components of the Elastic Stack run on hardware that is optimized for their use case. Documentation will be available when we release the new functionality in full.

If you want to learn more about the high-level features that flexible deployment configurations will bring, see Elastic Unveils a Record Number of New Features and Previews Future Developments at Elastic{ON} 2018.

Service release: June 19, 2018
