Support for SAML and multi-factor authentication released

Two important security features are now available for Elasticsearch Add-On for Heroku:

  • SAML Support for Elasticsearch and Kibana. To secure your Elasticsearch clusters and Kibana instances, you can now use a Security Assertion Markup Language (SAML) identity provider (IdP) for cross-domain, single sign-on authentication. Learn More …
  • Multi-factor Authentication (MFA) Support. To provide another layer of security to your Elastic Cloud account, you can now enable multi-factor authentication using a token, that can be enabled with Google Authenticator or SMS. Learn More …

Service release: October 29, 2018 (Milestone 12)
