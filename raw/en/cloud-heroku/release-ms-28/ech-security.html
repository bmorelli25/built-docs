Securing your deployment

Elasticsearch Add-On for Heroku supports most of the security features that are part of the Elastic Stack. These features are designed to:

  • Prevent unauthorized access with password protection and role-based access control:

    • Reset the elastic user password.
    • Manage your users on different versions.
    • Use third-party authentication providers and services like SAML, OpenID Connect, or Kerberos.
  • Preserve the integrity of your data with message authentication and SSL/TLS encryption.
  • Secure your settings with the Elasticsearch keystore.
