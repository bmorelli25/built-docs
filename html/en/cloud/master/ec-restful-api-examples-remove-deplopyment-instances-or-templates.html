Remove unwanted deployment templates and instance configurations

When you created your deployment and find that you created instance configurations or deployment templates that you no longer need, you can use these steps to delete them through the RESTful API. Alternatively, you can also use the UI to delete these artifacts.

Before you begin

Deleting instance configurations or deployment templates does not affect existing deployments that got created from these artifacts, but you should still make sure that you no longer need these artifacts, as there is no undo function.

Steps

To remove an instance configuration:

  1. List the available instance configurations to find out the ID for the instance configuration you want to delete: 

    curl -X POST -u $USER:$PASSWORD https://$CLUSTER_ID.REGION.PLATFORM.DOMAIN:9243
/api/v1/platform/configuration/instances

    A user with sufficient privileges

    The password for the user

    The cluster ID that is unique to your cluster

    The geographic region within your cloud platform. For example, us-east-1 or us-central1

    The cloud platform, either aws for Amazon Web Services or gcp for Google Cloud Platform

    The domain name. For example, cloud.es.io or found.io

  2. Delete the instance configuration using the ID: 

    curl -X POST -u $USER:$PASSWORD https://$CLUSTER_ID.$REGION.$PLATFORM.$DOMAIN:9243/api/v1/platform/configuration/instances/$INSTANCE_CONFIGURATION_ID

    The instance configuration ID as returned by the instance_configuration_id field

To remove a deployment template through the RESTful API:

  1. List the available deployment templates to find out the ID: 

    curl -X POST -u $USER:$PASSWORD https://$CLUSTER_ID.R$EGION.$PLATFORM.$DOMAIN:9243/api/v1/platform/configuration/templates/deployments

  2. Delete the deployment templates using the ID: 

    curl -X POST -u $USER:$PASSWORD https://$CLUSTER_ID.$REGION.$PLATFORM.$DOMAIN:9243
/api/v1/platform/configuration/templates/deployments/$DEPLOYMENT_TEMPLATE_ID

    The deployment template ID as returned by the id field
